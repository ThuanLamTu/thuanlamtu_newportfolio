# HƯỚNG DẪN KÍCH HOẠT TRANG PROFILE GITHUB ẤN TƯỢNG

Để hiển thị giao diện giới thiệu cá nhân tuyệt đẹp này trên trang chủ GitHub của bạn (như liên kết `github.com/ten-cua-ban`), hãy làm theo các bước sau:

1. Đăng nhập vào GitHub.
2. Tạo một **Kho chứa mới (New Repository)**.
3. Ở phần **Repository name**, hãy nhập **chính xác tên tài khoản GitHub (Username)** của bạn. 
   *(Ví dụ: Nếu tài khoản của bạn là `duc-dev`, tên repo phải là `duc-dev`).*
4. Hệ thống sẽ hiển thị một thông báo đặc biệt: *"You found a secret!..."* cho biết đây là kho chứa đặc biệt để làm Profile README.
5. Để chế độ **Public** và tick chọn **Add a README file**.
6. Sau khi tạo xong, hãy copy toàn bộ nội dung của phần **[MÃ NGUỒN README.MD]** ở dưới đây, nhấn chỉnh sửa file `README.md` trong repo đó, dán đè lên rồi nhấn **Commit changes**.
